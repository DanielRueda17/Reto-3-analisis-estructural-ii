# Fuente de secciones HSS rectangulares AISC

La fuente vigente para secciones es el archivo entregado por el docente:

```text
catalogo_AISC.pdf
```

El catalogo indica:

```text
ASTM A1085/A1085M Rectangular HSS
```

Por decision del equipo, se usaran perfiles HSS rectangulares de este catalogo, aunque el enunciado mencione tubulares cuadrados. Esta decision corrige la tabla reducida previa de HSS cuadrados, que quedo archivada como no vigente.

La tabla `secciones_aisc_hss_rectangular.csv` es una base inicial transcrita del catalogo del docente, pagina 2. Debe ampliarse si el diseno requiere perfiles fuera de los grupos incluidos.

Unidades originales del catalogo:

- dimensiones: in
- peso: lb/ft
- area: in2
- inercias: in4
- modulos: in3
- radio de giro: in
